# GSI A2 Study App

Next.js study app for GSI A2 practice tests. Import questions from Preparatic, take random tests, and track progress with Supabase.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Fill in Supabase credentials in .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
